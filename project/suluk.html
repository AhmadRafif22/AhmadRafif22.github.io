Merupakan project web yang dibangun oleh 2 orang. suluk book store merupakan website e-commerce yang menjual buku, terdiri dari tampilan admin dan pelanggan. admin dapat mengelola toko, mulai dari data buku, costumer, dan data transaksi, pelanggan dapat mencari buku dan memasukkannya dalam keranjang, kemudian melakukan checkout dan mengecek status transaksi. web ini dibangun manggunakan Framework Laravel.
