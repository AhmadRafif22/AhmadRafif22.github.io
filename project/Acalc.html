Merupakan project web yang dibangun oleh 3 orang. Project ini dibangun menggunakan PHP native dan Bootstrap 4 sebagai framework front-end-nya. Fitur utama dari project ini adalah fitur kalkulator scientific yang memungkinkan pengguna untuk melakukan komputasi matematika yang lebih kompleks seperti ln, trigonometri, dan fungsi-fungsi tingkat tinggi. Fitur login dan register juga tersedia sebagai fitur tambahan untuk memberikan keamanan dan akses yang lebih teratur bagi pengguna.
